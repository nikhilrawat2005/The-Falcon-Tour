========================================================================
                     FIREBASE CONFIGURATION GUIDE
========================================================================

Follow these steps to set up Firebase for The Falcon Tour website:

1. CREATE A FIREBASE PROJECT:
   - Go to the Firebase Console: https://console.firebase.google.com/
   - Click "Add Project" and name it "The-Falcon-Tour" (or any name you prefer).
   - Configure or disable Google Analytics as desired, then click "Create Project".

2. REGISTER A WEB APP:
   - In the Firebase project overview, click the Web icon (</>) to register a web application.
   - Enter an App nickname (e.g., "Falcon Web App").
   - Click "Register app".
   - You will see a `firebaseConfig` object containing your unique keys:
     const firebaseConfig = {
       apiKey: "YOUR_API_KEY",
       authDomain: "YOUR_PROJECT_ID.firebaseapp.com",
       projectId: "YOUR_PROJECT_ID",
       storageBucket: "YOUR_PROJECT_ID.firebasestorage.app",
       messagingSenderId: "YOUR_SENDER_ID",
       appId: "YOUR_APP_ID"
     };
   - Copy these values and paste them into the config placeholder inside:
     `assets/js/firebase-config.js`

3. ENABLE FIREBASE AUTHENTICATION:
   - In the Firebase Console sidebar, click "Build" > "Authentication".
   - Click "Get started".
   - Go to the "Sign-in method" tab.
   - Click "Add new provider" and select "Email/Password".
   - Enable "Email/Password" and click "Save".

4. CREATE A FIRESTORE DATABASE:
   - In the Firebase Console sidebar, click "Build" > "Firestore Database".
   - Click "Create database".
   - Select your database location and click "Next".
   - Select "Start in test mode" (or locked mode) and click "Create".
   - GO TO THE "Rules" TAB and paste the following security rules to ensure safety:

     rules_version = '2';
     service cloud.firestore {
       match /databases/{database}/documents {
         // Users collection rules
         match /users/{userId} {
           allow read: if request.auth != null;
           allow write: if request.auth != null && request.auth.uid == userId;
         }
         // Submissions collection rules
         match /submissions/{subId} {
           // Allow anyone to submit a contact form
           allow write: if true;
           // Only admin (email: nikhil2005114@gmail.com) can read submissions
           allow read: if request.auth != null && request.auth.token.email == 'nikhil2005114@gmail.com';
         }
       }
     }
   - Click "Publish".

Now your Firebase setup is ready and the frontend will automatically connect to it!
